    get_settings_filename ¶

    get_settings_filename(filename: Path) -> Path

    Determine path of settings file.

    Search strategy:

    1. 'DICOM_QR_SETTINGS' environment value
    2. 'settings.json' in current directory
    3. '~/.config/dicom_qr/settings.json'
